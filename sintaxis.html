Sintaxis Javascript

Ubicación

JavaScript se puede agregar de tres formas principales a una página web:

1. Etiquetas: Se puede insertar el código JavaScript dentro de las etiquetas <script> en el archivo HTML.

2. Web Page: Se puede incluir JavaScript directamente en las páginas web mediante un enlace a un archivo externo .js.

3. Archivo JS: Se puede tener un archivo JavaScript separado con extensión .js que contiene el código y se vincula a la página HTML.

Comentarios

Los comentarios en JavaScript son utilizados para dejar notas dentro del código. Los comentarios no son ejecutados. Hay dos tipos de comentarios:

  • Comentario de una sola línea: Se inicia con //
  • Comentario de varias líneas: Se inicia con /* y termina con */

Ejemplo:


    // Este es un comentario de una sola línea
    
    /* 
    Este es un comentario de múltiples líneas
    que puede cubrir varias líneas de código
    */

Script

El código JavaScript generalmente se coloca dentro de las etiquetas <script> en un archivo HTML.

Ejemplo básico de un script:


    <script>
      alert('¡Hola, Mundo!');
    </script>

El código anterior muestra un mensaje emergente en el navegador.

Modo Estricto

El modo estricto de JavaScript permite a los desarrolladores escribir un código más seguro y menos propenso a errores. Se activa añadiendo 'use strict'; al inicio de un archivo o función.

Ejemplo de uso del modo estricto:


    'use strict';
    x = 10; // Esto causará un error, porque "x" no ha sido declarada previamente.

En el ejemplo, sin el modo estricto, se crearía una variable global sin declarar, pero con el modo estricto se genera un error, ayudando a evitar errores y problemas de seguridad.
